# SiteComunidade - Desenvolvimento de Aplicação Web com Backend em Python

Heroku: https://ramalho-sitecomunidade.herokuapp.com/

Site desenvolvido através do curso de Python da Hashtag Programação, com o instrutor João Lira.

- Utilização do Framework Bootstrap para o FrontEnd
- Utilização do Framework Flask para o BackEnd em Python
- Utilização do ORM Sqlalchemy como banco de dados
